# Folio
Short description and motivation.

## Usage
Copy `redactor.js` and `redactor.css` to `test/dummy/vendor/assets/redactor`. 

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'folio'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ rails generate folio:install
```
